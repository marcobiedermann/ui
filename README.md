## Installation

```sh
npm install
```

## Usage

### Storybook

Launch Storybook development server with hot module replacement.

```sh
npm run develop
```

Build Storybook for production into `docs` folder.

```sh
npm run build
```

### Testing

Run all tests and linting.

```sh
npm test
```

Run CSS and JavaScript linting.

```sh
npm run lint
```

Run CSS linting.

```sh
npm run lint:css
```

Run JavaScript linting.

```sh
npm run lint:js
```

```sh
npm run publish
```
